1.What is DOM and What is the purpose of DOM ?

DOM means the Document Object Model. DOM is a programming interface for web documents. It represent the web page and the document as nodes and objects so that programming languages can interact with the page and change the document structure, style and content.

2.How will you select HTML element using DOM? Name the DOM methods?

I will select HTML element by using DOM selecting methods. Some selecting methods getElementById, getElementsByClassName, getElementsByTagName, getElementsByName, querySelector and querySelectorAll.

3.What is event bubble ?

Event bubble is a type of DOM event propagation. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

4.What is a callback function and why will you use it ?

A callback function is a function that is to be executed after another function has finished executing. Callback functions are needed because many JavaScript actions are asynchronous. A callback's purpose is to execute code in response to an event.
